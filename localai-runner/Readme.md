# LocalAI Plugin

Simple plugin to install and manage LocalAI on your system.

## Quick Start

### Make scripts executable:
```bash
chmod +x *.sh
```

1. Install service:

```bash
./install.sh
```

2. Start the service:

```bash
./start.sh
```
3. Stop the service:

```bash
./stop.sh
```
4. Uninstall the service:

```bash
./uninstall.sh
```
